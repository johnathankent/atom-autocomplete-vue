# Vue.js autocomplete and snippets for Atom

Vue.js 2.0+

## Autocomplete

### Supported tags

* partial
* slot

### Supported Directives

* v-text
* v-html
* v-if
* v-else-if
* v-show
* v-else
* v-for
* v-on
* v-bind
* v-model
* v-pre
* v-cloak
* v-once

### Built-In Components [TODO]

* component
* transition
* transition-group

## Snippets

> Works **only** with `*.vue` files.

Just type `vtemplate`, `vscript` or `vstyle` to generate the respective sections.
